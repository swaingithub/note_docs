# Assembly Language — Complete Course

A step-by-step, hands-on Assembly course for absolute beginners. We start from "what is a CPU" and end by writing a small real program you run on your own machine (and in the browser playground). Follow the lessons in order, type every snippet, tick the checklist.

> **Target ISA:** x86-64 NASM (Linux). A browser **RISC-V playground** is included so you can run code without installing a toolchain.
> **Difficulty:** Hard &nbsp;•&nbsp; **Time:** ~8–10 hours &nbsp;•&nbsp; **Style:** Udemy-style, project-based.
