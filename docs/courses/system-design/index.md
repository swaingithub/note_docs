# System Design — Complete Course

Design scalable, reliable, and maintainable systems. This course goes from the **building blocks** (networking, caching, databases) to **architecture patterns** (microservices, queues, resilience) and ends with **real end-to-end designs** and an **interview framework**. References: *Designing Data-Intensive Applications*, *System Design Primer*, *MIT 6.824*.

> **Style:** Udemy-style, interview-focused. Each lesson has diagrams (ASCII), exercises, flashcards, and a checklist.
> **Best paired with:** [AWS course](/courses/aws) (you'll recognize the services) and [Databases](/courses/databases).
