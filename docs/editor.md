# ✏️ Live Editor

Edit Markdown on the left, see the rendered preview on the right. **Everything is saved automatically in your browser** (localStorage) — close the tab and come back, your notes are still here.

<LiveEditor doc="scratch" title="My Scratch Notes" />

## Features

- **Live preview** — rendered as you type
- **Autosave** — no save button needed
- **Export** — download as `.md` to move notes into a docs page
- **Import** — load an existing `.md` file
- **Per-note storage** — each `doc` id keeps its own content

## How to make a note permanent

Use the editor to draft, then **Export .md** and drop the file into `docs/`. Or paste the content directly into any page under `docs/languages/` or `docs/courses/`.
